# Лабораторная работа №1 Вариант 9
## Предметная область
## Объект (Что является сущностью?)
Реализованный класс Apartment хранит в себе данные о квартире: площадь, цена аренды, адрес, срок аренды. Также логическое состояние — арендована или свободна.
## Свойства объекта (Какие у него атрибуты?)
### Атрибуты класса
_total_apartments — общее количество созданных экземпляров квартир.
### Атрибуты объекта (экземпляра)
_area — площадь квартиры (м²)
_price — полная стоимость аренды (руб.)
_address — адрес квартиры (строка)
_rent_duration — срок аренды в месяцах
_is_rented — флаг: арендована (True) или свободна (False)
## Инварианты
area всегда положительное число 
price всегда положительное число 
address не может быть пустой строкой или состоять из пробелов
rent_duration целое положительное число 
_is_rented == True, запрещено изменять area, price, address, rent_duration
нельзя сдать в аренду уже арендованную квартиру, и нельзя освободить свободную
## Метод __eq__ (Что значит «равенство»?)
Метод __eq__ по сути является скрытой перегрузкой оператора сравнения =
Задаем критерии, по которым будем сравнивать два объекта одного класса
В моем случае квартиры считаются равными, если совпадают address и area
## Класс состояний (Есть ли состояние?)
Присутствует состояние аренды квартиры, которое регулируется булевым значением флага _is_rented
Свободна (_is_rented == False) — квартира доступна для аренды, все атрибуты можно изменять
Арендована (_is_rented == True) — изменение атрибутов запрещено (кроме вызова vacate())
## Методы
### Конструктор
__init__(area, price, address, rent_duration) — конструктор, валидирует входные данные, увеличивает счётчик _total_apartments, начальное состояние — свободна
### Геттеры и сеттеры через @property
@property area, price, address, rent_duration, is_rented — для чтения
@area.setter, @price.setter, @address.setter, @rent_duration.setter — для изменения, но с проверкой если _is_rented == True — RuntimeError
### Методы изменения состояния
rent() — переводит квартиру в состояние арендована (если уже арендована — ошибка)
vacate() — освобождает квартиру (если свободна — ошибка)
### Бизнес-методы
calculate_monthly_payment() → float — ежемесячный платёж за квартиру, цена/время аренды
is_expensive(threshold) → bool — дороже ли стоимость аренды заданного пользователем порога
### Методы для отладки
__str__() — «красивое» представление 
__repr__() — техническое представление
### Метод класса
get_total_apartments() — возвращает кол-во арендованных квартир
## @property — что даёт?
динамическое вычисление
защита данных (позволяет добавить проверку)
можно обращаться к атрибуту без скобок
чтение _name и подобных
## __repr__
Выводит данные для разраба, так скажем некрасиво
Возвращает строковое представление объекта
